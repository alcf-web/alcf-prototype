# alcf-prototype

## Installation instructions
1. Install virtual env for python
`$ pip install virtualenv`
2. Create a virtual env directory
`$ virtualenv alcf-env`
3. cd into the directory
`$ cd alcf-env`
4. Install wagtail
`$ pip install wagtail`
5. git clone alcf repository
6. Run the server
`$ python manage.py runserver`
